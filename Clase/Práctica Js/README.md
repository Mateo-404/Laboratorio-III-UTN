# Ejercicio 1
Crea una función que tome dos números como argumentos y devuelva el mayor de ellos.
 
# Ejercicio 2
Escribe una función que tome un array de números como argumento y devuelva la suma de los elementos del array.
 
# Ejercicio 3

# Ejercicio 4
